# API Endpoints

This document provides information about the available API endpoints for TaskMaster.

## GET /tasks

Retrieve a list of all tasks.

```http
GET /tasks
```

## POST /tasks
Create a new task.

```http
POST /tasks
```

## PUT /tasks/:id
Update an existing task.

```http
PUT /tasks/:id
```

## DELETE /tasks/:id
Delete a task.

```http
DELETE /tasks/:id
```
