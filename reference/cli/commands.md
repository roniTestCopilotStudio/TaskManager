# CLI Commands

This document provides information about the available CLI commands for TaskMaster.

## taskmaster init

Initialize a new TaskMaster project.

```bash
taskmaster init
```

## taskmaster start
Start the TaskMaster application.

```bash
taskmaster start
```

## taskmaster build
Build the TaskMaster application for production.

```bash
taskmaster build
```

## taskmaster test
Run the test suite.

```bash
taskmaster test
```
